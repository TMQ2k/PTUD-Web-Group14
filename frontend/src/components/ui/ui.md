# UI Components

Các thành phần giao diện người dùng (UI Components) là các phần tử xây dựng giao diện người dùng trong ứng dụng của bạn. Chúng bao gồm các nút, biểu mẫu, hộp thoại, thanh điều hướng và nhiều hơn nữa. Sử dụng các thành phần UI giúp tạo ra trải nghiệm người dùng nhất quán và dễ sử dụng.

## Ví dụ

- Button.jsx
- Modal.jsx
- Navbar.jsx
- FormInput.jsx
